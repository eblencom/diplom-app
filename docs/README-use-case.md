# Диаграмма вариантов использования

Файл: `use-case-diplomapp.puml` — формат **PlantUML** (актёры отображаются как «человечки»).

## Как получить картинку

1. Откройте [plantuml.com/plantuml](https://www.plantuml.com/plantuml/uml/) или [planttext.com](https://www.planttext.com/).
2. Вставьте содержимое `use-case-diplomapp.puml` в редактор.
3. Сохраните PNG/SVG для пояснительной записки.

Локально: плагин PlantUML в VS Code / IntelliJ, или CLI `plantuml` при установленом Java и Graphviz.

В актуальной версии диаграмма **компактная** (8 вариантов), три актёра выстроены **в ряд слева — центр — справа** над границей системы (`-[hidden]right-` между соседними актёрами).

## Про «GoKanban»

Отдельного общепринятого сервиса с таким названием для UML нет. Если на вашем курсе требуется **конкретный сайт** — вставьте туда код, **если сайт поддерживает PlantUML**; иначе экспортируйте PNG из PlantUML и вставьте в доску Kanban как вложение.
